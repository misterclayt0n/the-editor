# The editor
### Motivation
I love neovim, and wanted to create my own!

### Usage
1. Install [zig](https://ziglang.org/)

2. Clone the repository
```zsh
git clone https://github.com/misterclayt0n/the-editor
```

3. Build the-editor
```zsh
cd the-editor
zig build
```

4. Edit your files
```zsh
./zig-out/bin/the-editor <FILE>
```
