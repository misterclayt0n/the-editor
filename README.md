# The editor
### Motivation
I love neovim, and wanted to create my own!

### Usage
1. Install [rust](https://www.rust-lang.org/)

2. Clone the repository
```zsh
git clone https://github.com/misterclayt0n/the-editor
```

3. Build the-editor
```zsh
cd the-editor
cargo build
```

4. Edit your files
```zsh
./target/debug/editor <FILE>
```
